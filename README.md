## Overview

Aureus is a Roman Empire–inspired multi-chain cryptocurrency treasury platform engineered for modern digital asset management.

Designed with institutional aesthetics and trader-focused functionality, Aureus provides a unified environment for managing cryptocurrencies, monitoring blockchain activity, executing transactions, and tracking portfolio performance across blockchain ecosystems.

The platform combines high-performance blockchain infrastructure with an interface inspired by the authority and precision of the Roman treasury system.

## Installation

Clone the repository:

```bash
git clone https://github.com/SheerWill007/aureus.git
```

Navigate into the project:

```bash
cd aureus
```

Install dependencies:

```bash
npm install
```

Configure environment variables:

```bash
cp .env.example .env.local
```

Run development server:

```bash
npm run dev
```

Application launches locally:

```
http://localhost:3000
```
